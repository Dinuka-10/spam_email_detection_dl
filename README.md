# 🤖 Spam Email Detection (ANN)

## 📌 About

This is my **first Artificial Neural Network (ANN) project**.
It classifies emails as **Spam** or **Not Spam (Ham)** using deep learning.

---

## 🛠️ Technologies

* Python
* PyTorch
* Scikit-learn
* Pandas & NumPy
* NLP (TF-IDF / CountVectorizer)

---

## ⚙️ How It Works

* Convert email text into numerical features
* Train an ANN model
* Predict whether the email is spam or not

---

## ⚠️ Note

❌ Trained model files (`.pkl`, `.pt`) are **not uploaded**

To use this project:

* Train the model using the code
* Save the model inside the `model/` folder
